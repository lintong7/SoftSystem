**项目简介：**
软件下载系统主要模块：
	（1）登录注册模块：实现了登录、注册功能
	（2）管理员模块：管理员列表、权限、激活、删除、修改管理员信息等操作。
	（3）软件模块：软件列表、软件上传、下载、修改软件信息等操作。
	（4）日志模块：主要记录系统的各种操作信息。
		技术：C3P0+dbUtils、JSP+Servlet、jQuery、Ajax、Bootstrap等技术。
